HTML & CSS Basics Summary

  • HTML Summary

    HTML (HyperText Markup Language) is used to define our page content, structure and meaning. You don't use it for styling purposes. Use CSS for that instead!

    HTML uses "elemnents" to describe (annotate) content

    HTML elemnents typically have an opening tag, content and then a closing tag

    You can also have void (empty) elemnents like images You can also configure elemnents with attributes

    There's a long list of available elemnents but you'll gain experience over time, no worries. Learn more about all available HTML elemnents on the MDN HTML elemnents reference

  • CSS Summary

    CSS (Cascading Style Sheets) is used for styling your page content.

    Styles are assigned via property-value pairs You can assign styles via the "style" attributes

    To avoid code duplication, you typically use global styles (e.g. via the "style" elemnents) instead

    Alternatively, you can work with external stylesheet files which you "link" to

    When working with CSS, concepts like "inheritance", "specificity" and the "box model" should be understool

    Learn more about all available CSS properties and values on the MDN CSS property reference
